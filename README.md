# Expert-Routed Harmful Speech Detection App (UI)

Frontend UI for SERML built with Vite + React + Tailwind + Recharts.

## Setup

```bash
npm install
cp .env.example .env
```

Set backend URL in `.env`:

```bash
VITE_API_BASE_URL=https://<your-backend-url>
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
